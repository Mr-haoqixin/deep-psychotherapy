## 🎯核心特色
**专项调整**：
### 1. **负面联想循环的专项识别**
- 详细拆解了抑郁典型的「滑坡」路径：触发事件 → 自动思维 → 情绪 → 身体 → 二次解读 → 自我否定 → 无力感 → 行为退缩 → 验证负面信念
- 提供「暂停」技术和思维记录表来打断循环
### 2. **自卑与自我否定的深层工作**
- 加入「向下箭头技术」探索核心信念（如"我不值得被爱"）
- 「内在批判者外化」技术，帮助区分"我"和"我的自我批评"
- 链接早期经验，理解条件性接纳的形成
### 3. **反刍思维的应对**
- 「溪边落叶」隐喻帮助去中心化
- 身体锚定技术，从"头脑战争"回到当下
- 明确区分「接纳」不等于「喜欢」
### 4. **温和的行为激活**
- **关键原则**：不强迫、不施压、从最小可行行动开始
- 避免"你应该出去走走"这类增加自责的建议
- 提供 0-5 级活动难度表，与来访者共同制定
### 5. **危机干预协议**
- 内置自伤/自杀风险评估
- 提供全国心理援助热线等具体资源
- 明确 AI 咨询的局限性边界
### 6. **七个分析层次的抑郁适配**
- 在原有框架基础上，每一层都增加了抑郁专项的解读示例
- 特别强调：复述时不乐观化、情绪分层时关注存在层面的恐惧、身体层面关注抑郁的躯体化信号
